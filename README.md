- `borales/nginx-php:latest-dev`, `borales/nginx-php:7.0-dev` - PHP 7.0 with XDebug installed
- `borales/nginx-php:7.1` - PHP 7.1
- `borales/nginx-php:7.1-dev` - PHP 7.1 with XDebug installed
